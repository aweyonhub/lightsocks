LightSocks
==========

A socks5 proxy server for fun.

Requirements
-----------

- PyCrypto <https://www.dlitz.net/software/pycrypto/>

How to Run
----------

`./server.py` for server side and `./local.py` for local side

**NOTE**: for security, do not use simple KEY.
